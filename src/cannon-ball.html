Cannon Ball

Web Steganography Medium

Challenge Information

  • Challenge Name: Cannon Ball
  • Category: Web, Steganography
  • URL: https://cannon-ball-three.vercel.app/
  • Description: "There is something wrong with this website, but I can't put a finger on it."
  • Flag: EOF{F0und_!t}

Solution Methodology

Phase 1: Initial Website Reconnaissance

curl -s https://cannon-ball-three.vercel.app/ > page.html
cat page.html | grep -i "EOF\|flag\|hidden"

Key Findings:

  • HTML comment: <!--RCnYi2VTt9yXtWnnmzOPL-->
  • Low opacity image element with id f00lish-stuff (opacity: 0.01)
  • Another image with id secret-stuff (opacity: 0.2)
  • Multiple elements with very low opacity values designed to be nearly invisible

Phase 2: JavaScript Analysis

grep -E "opacity|display.*none|visibility.*hidden" page.html

Phase 3: Hidden Resources Discovery

Found base64-encoded data URIs and hidden image sources in the HTML. Extract these resources for analysis.

Phase 4: Image Steganography

After downloading hidden images, analyze them using steganography tools:

# LSB steganography analysis
zsteg hidden_image.png

# Metadata analysis
exiftool hidden_image.png

# String extraction
strings hidden_image.png | grep "EOF"

Phase 5: Flag Extraction

The flag was hidden using LSB (Least Significant Bit) steganography in one of the PNG images.

Flag: EOF{F0und_!t}

Key Insights

Multi-Stage Challenge

  1. Web reconnaissance and hidden resource discovery
  2. Base64-encoded data URIs analysis
  3. Multiple website layers with misdirection
  4. LSB steganography in PNG images
  5. Game interface as additional misdirection

Tools Used

  • curl - For fetching web pages
  • zsteg - For LSB steganography analysis
  • tesseract - For OCR if needed
  • base64 - For decoding data URIs
  • exiftool - For metadata analysis
  • Browser DevTools - For inspecting hidden elements

Key Techniques

  • Inspecting CSS for hidden elements (opacity, display, visibility)
  • Analyzing base64-encoded images in data URIs
  • Understanding LSB steganography in images
  • Web scraping and automated analysis
